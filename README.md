# Flow
🌊 Flow Programming Language

High-Level Simplicity. Low-Level Power.

Flow is a hybrid programming language designed for natural readability without sacrificing hardware control. It allows developers to write code using intuitive, English-like syntax that executes with the efficiency of lower-level systems.

🚀 Key Features

Human-Centric Syntax: Write logic in plain English (e.g., Create a Number called Power).

Blazing Performance: Optimized via the Flow C-Bridge for native execution speeds.

Hardware Control: Direct system kernel communication for advanced resource management.

FlowPad IDE: A dedicated desktop development environment with live syntax highlighting and an integrated debug shell.

💻 Quick Start

Installation

Ensure you have Python 3.8+ installed.

Clone this repository:

git clone [https://github.com/imredavid64-glitch/Flow.git](https://github.com/imredavid64-glitch/Flow.git)


Run the IDE:

python flowpad.py
